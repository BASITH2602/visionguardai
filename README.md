# \# VisionGuard AI 👁️🤖

# 

# VisionGuard AI is a real-time AI-powered eye wellness and attention monitoring system built using Flask, OpenCV, MediaPipe, and JavaScript.

# 

# \## 🚀 Features

# 

# \- Real-time face tracking

# \- Blink detection

# \- Attention monitoring

# \- Eye strain detection

# \- Distance monitoring

# \- Low light detection

# \- AI voice alerts

# \- Vision testing

# \- Color blindness testing

# \- Live behavioral analytics

# \- Premium AI PDF reports

# \- Webcam snapshot analysis

#

#

# 

# VisionGuard AI analyzes:

# \- User attention

# \- Eye fatigue
