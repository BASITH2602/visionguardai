VisionGuard AI 👁️🤖

VisionGuard AI is a real-time AI-powered eye wellness and attention monitoring system built using Flask, OpenCV, MediaPipe, and JavaScript.

🚀 Features

#  Real-time face tracking
#  Blink detection
#  Attention monitoring
#  Eye strain detection
#  Distance monitoring
#  Low light detection
#  AI voice alerts
#  Vision testing
#  Color blindness testing
#  Live behavioral analytics
#  Premium AI PDF reports
#  Webcam snapshot analysis

🧠 AI Capabilities

 VisionGuard AI analyzes:

#  User attention
#  Blink behavior
#  Eye fatigue
#  Visual wellness
#  Distance from screen
#  Environmental lighting
#  The system generates intelligent recommendations and futuristic diagnostic reports.

📊 Tech Stack

- Python
- Flask
- OpenCV
- MediaPipe FaceMesh
- HTML
- CSS
- JavaScript
- Chart.js
- jsPDF
- HTML2Canvas

📄 Premium Report Features

#  AI wellness summary
#  Recommendations
