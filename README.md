# VisionGuard AI

VisionGuard AI is a real-time AI-powered eye wellness and attention monitoring system built using Flask, OpenCV, MediaPipe, and JavaScript.

## Features

- Real-time face tracking
- Blink detection
- Attention tracking
- Eye strain monitoring
- Distance detection
- Voice alerts
- Vision testing
- Color blindness testing
- AI analytics dashboard
- Premium 4-page PDF reports

## Technologies Used

- Python
- Flask
- OpenCV
- MediaPipe
- HTML/CSS/JavaScript
- jsPDF
- Chart.js
